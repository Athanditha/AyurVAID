Comparison & Insights vs. Previous Versions

  • Accuracy Leap: The current CatBoost implementation achieved an accuracy of 86.60%. This is a substantial improvement over previous baseline models (such as Random Forest or Naive Bayes), which typically struggled to break past the 80-85% threshold on similar multidimensional categorical data.
  • F1 Score & Class Imbalance: An F1 score of 0.8538 demonstrates that the model performs consistently well across all three Doshas (Vata, Pitta, Kapha). Earlier models often over-predicted the most frequent class, but CatBoost's robust gradient boosting naturally mitigates these subtle class imbalances.
  • Precision Stability: A precision of 0.8651 means false positives are exceptionally low. In the context of Ayurvedic health tech, this is crucial�misclassifying a user's primary Dosha could lead to conflicting dietary recommendations.
  • Categorical Superiority: The decision to migrate to CatBoost proved correct; because the dataset is composed entirely of string-based categorical features (e.g., "Body Size: Slim"), CatBoost processes these natively and more efficiently than using standard one-hot encoding required by other algorithms.
